## 🎯 Milestone 3 – Dashboard Development & Interactive Analytics

This milestone focuses on building a fully interactive **Power BI dashboard** that enables side-by-side military comparison between two countries. The result is a dynamic **Compare Powers Dashboard** that transforms raw military data into actionable strategic insights.

---

### ⚔️ Compare Powers Dashboard
An advanced interactive dashboard that allows users to analyze and compare military strength between two nations across multiple dimensions.

#### ❓ What This Dashboard Answers
* **Overall Strength:** Which country is stronger overall?
* **Domain Analysis:** How do two countries differ in air, land, and sea power?
* **Fiscal Comparison:** How does defense spending compare between nations?

---

### 🧠 Core Features

#### 🔄 Interactive Country Selection
* Select **Country A** and **Country B** dynamically.
* Dataset updates instantly based on selection.

#### 📊 KPI Cards (Per Country)
Each selected country displays key strategic indicators:
* 🥇 **Power Index Rank**
* 👥 **Assets per Capita**
* 💰 **Budget-to-GDP Ratio**

#### ⚖️ Military Comparison Metrics
* **✈️ Air Power:** Total Aircraft & Fighter Aircraft.
* **🚜 Land Power:** Tanks.
* **🚢 Naval Power:** Naval Fleet Strength.
* **👥 Manpower & Budget:** Visual comparison of total personnel and defense spending.

#### 🌍 Visual Comparison
* Side-by-side country maps.
* Clear visual contrast using charts and KPIs.
* Easy identification of strengths and weaknesses.

#### 🔀 Dashboard Navigation
* Seamless navigation to the next module: **Coalition Builder Dashboard**.

---

### 📊 Key Visuals Used
* **Bar Charts:** Aircraft, Navy, and Manpower comparison.
* **Donut Chart:** Budget distribution.
* **KPI Cards:** Key strategic indicators.
* **Map Visuals:** Geographic country highlighting.
* **Slicers:** Dynamic country selection.
