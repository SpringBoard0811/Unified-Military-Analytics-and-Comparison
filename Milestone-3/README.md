# Milestone 3: Dashboard Development

## Overview

Milestone 3 focuses on building interactive dashboards that visualize the processed military dataset. These dashboards allow users to explore global military strength through dynamic charts, filters, and comparisons.

## Objectives

* Build multiple dashboards for military analysis
* Enable comparison between countries
* Provide interactive filtering and navigation

## Dashboard Modules

### Quick Stats Dashboard

* Displays top countries based on Power Index
* Shows summary statistics of global military capabilities
* Includes filters such as region, continent, and alliance

### Nation Overview Dashboard

* Provides a detailed profile of an individual country
* Visualizes manpower, aircraft, naval assets, and defense budget
* Uses charts and tooltips for better insights

### Compare Powers Dashboard

* Allows side-by-side comparison of two countries
* Displays differences in military resources and KPIs

### Coalition Builder Dashboard

* Allows selection of multiple countries
* Calculates the combined strength of selected nations
* Compares coalition strength with other countries

## Deliverables

* Dashboard application prototype
* Completed dashboard tabs for analysis

## Expected Outcome

A fully interactive dashboard environment that allows users to analyze and compare military power across countries using visual analytics.
