# 📊 Milestone 3 – Interactive Military Analytics Dashboard

## 🌐 Global Military Insights Platform

Milestone 3 focuses on developing **interactive dashboards using Power BI** to visualize and analyze global military data.

Using the **KPI-enhanced dataset created in Milestone 2**, this stage converts processed military data into **visual insights** that allow users to explore, analyze, and compare the military capabilities of different countries.

The dashboard system enables users to understand:

* 🌍 Global military power distribution
* 🪖 Defense capabilities of individual countries
* ⚔️ Strategic comparison between nations
* 🤝 Potential strength of military alliances

This milestone represents the **visual analytics layer** of the project.

---

# 🧭 Dashboard Modules

The Power BI application contains **four major dashboards**, each designed for a different level of military analysis.

---

## 🌍 Global Overview Dashboard

The **Global Overview Dashboard** provides a **high-level summary of military strength across countries worldwide**.

### Key Features

* Displays **Top 10 Countries by Power Index**
* Global **KPI summary cards**
* Interactive filtering based on:

  * Continent
  * Region
  * Military alliances

### Visualizations Used

* 📊 **Bar Chart** – Top 10 military powers
* 📌 **KPI Cards** – Key defense indicators
* 📈 **Regional comparison charts**

### Purpose

This dashboard helps users quickly **identify leading military powers and regional trends**.

---

## 🏳️ Country Military Profile Dashboard

The **Country Military Profile Dashboard** provides **detailed military statistics for a selected country**.

### Key Indicators

* Military Personnel
* Aircraft Strength
* Tank Inventory
* Naval Fleet
* Defense Budget
* Engineered KPI metrics

### Visual Components

* 📊 **Bar charts** for category comparisons
* 📡 **Radar chart** representing overall military capability
* 🔍 **Tooltips** displaying global ranking information

### Purpose

This dashboard allows users to analyze **a country’s military strength in detail** and understand how it compares globally.

---

## ⚔️ Military Comparison Dashboard

The **Military Comparison Dashboard** enables **side-by-side comparison of two countries**.

### Compared Metrics

* Military Manpower
* Aircraft Strength
* Tanks
* Naval Assets
* Defense Budget
* KPI indicators

### Interaction

Users can dynamically select countries using **Power BI slicers or parameters**, allowing the dashboard to update instantly.

### Purpose

This dashboard helps identify **strategic advantages and capability differences between nations**.

---

## 🤝 Coalition Strength Dashboard

The **Coalition Strength Dashboard** simulates the **combined military capabilities of multiple countries**.

### Aggregated Metrics

* Total Military Personnel
* Total Aircraft
* Total Tanks
* Naval Assets
* Defense Budget

### Key Features

* Multi-country selection using filters
* Automatic aggregation of military metrics
* Coalition vs single-country comparison

### Purpose

This dashboard helps analyze **potential alliance strength and combined defense capabilities**.

---

# 🔗 Dashboard Navigation

All dashboards are integrated within **a single Power BI report**, allowing smooth navigation between analysis views.

### Navigation Flow

Quick Overview → Country Profile → Military Comparison → Coalition Strength

### Navigation Features

* Dashboard navigation buttons
* Shared filters across dashboards
* Interactive parameter controls
