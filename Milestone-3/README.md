# 🎯 Milestone 3 – Interactive Dashboard Development

This milestone focuses on developing the **complete interactive dashboard system** using Microsoft Power BI to analyze and compare global military power.

The goal is to transform the **engineered dataset and KPIs** from the previous milestone into a set of **interactive analytics dashboards** that allow users to explore military capabilities across countries.

The final dashboard system contains **four connected modules** that support global analysis, country-level insights, comparative analytics, and coalition simulations.

---

# ⚡ Quick Stats & Nation Overview

These dashboards provide a **high-level overview and detailed country-level analysis** of global military strength.

### Tasks

✔ Display the **Top 10 countries by Power Index**

✔ Add interactive filters for:

- 🌍 Region  
- 🌎 Continent  
- 🤝 Alliance  

✔ Create **KPI cards** for key indicators such as:

- Power Index Rank  
- Defense Budget  
- Aircraft Inventory  
- Active Personnel  

✔ Allow users to **select a country** and view its full military capability profile

✔ Visualize key metrics using charts such as:

- Bar charts  
- Radar charts  
- Comparative visualizations  

✔ Add **tooltips and contextual information** to show rankings and metric comparisons.

# ⚔️ Compare Powers & Coalition Builder

These dashboards focus on **comparative and scenario-based military analysis**.

### Tasks

✔ Enable **side-by-side comparison of two countries**

✔ Compare metrics such as:

- Military manpower  
- Aircraft inventory  
- Naval assets  
- Defense spending  
- Strategic KPIs

✔ Allow **multi-country selection** to simulate military coalitions

✔ Display **aggregated coalition metrics**, including:

- Combined manpower  
- Total aircraft  
- Naval fleet strength  
- Total defense budget

✔ Integrate dashboards using **navigation buttons, filters, and shared slicers** to create a unified dashboard experience.
