Milestone 4: Collision Builder
📌 Overview

Milestone 4 introduces the Collision Builder System, an advanced simulation module designed to model and analyze military conflict scenarios between nations or coalitions.

Building upon the dashboards from Milestone 3, this phase enables users to simulate potential clashes, evaluate relative strengths, and understand strategic outcomes based on combined military metrics.

🎯 Objective

To design an interactive system that:

Simulates conflicts between two or more entities
Evaluates military strength dynamically
Provides analytical insights into possible outcomes
