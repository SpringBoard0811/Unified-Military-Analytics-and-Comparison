# 📊 Milestone 2 – Power BI Dashboard (Nation Military Overview)

This milestone focuses on building the **first Power BI dashboard** for the project.
The goal of this dashboard is to provide a **quick overview of global military power and country-level military capabilities** using the cleaned dataset created in **Milestone 1**.


---

# 🎯 Objective

Create an **interactive Power BI dashboard** that provides a **high-level overview of global military strength** and allows users to explore military resources of different nations.

The dashboard highlights:

* Global military statistics
* Top military countries
* Defense budget comparison
* Military assets overview
* Aircraft, tanks, and naval fleet distribution

---

# 📂 Data Source

The dashboard uses the cleaned dataset produced in **Milestone 1**.

Dataset File:

```
military_cleaned.csv
```

The dataset contains structured military metrics for **145 countries**, including:

* ✈️ Total Aircraft
* 🛡️ Total Tanks
* 🚢 Naval Fleet
* 👥 Active Personnel
* 💰 Defense Budget

---

# 🛠 Tools & Technologies

* Power BI Desktop
* CSV Dataset
* Data Modeling
* Interactive Data Visualization

---

# 📊 Dashboard Overview

This dashboard provides a **quick global overview of military capabilities across countries** through multiple visualizations.

---

## 🌍 Quick Statistics

Summary metrics displayed on the left panel:

* **Total Countries Analyzed:** 145
* **Top Military Country:** United States
* **Global Defense Budget:** ~3 Trillion USD
* **Global Military Manpower:** ~24 Million

---

## 🏆 Top Military Countries

A bar chart showing countries with the highest military capabilities based on key metrics.

Countries include:

* United States
* Russia
* China
* India
* North Korea

---

## 💰 Defense Budget Comparison

A column chart comparing the **defense budgets of major military powers**.

This visualization highlights how military spending differs across countries.

---

## 🛡 Military Assets Comparison

A stacked bar chart comparing multiple military assets:

* Aircraft
* Tanks
* Naval Fleet

This provides an overall comparison of **military hardware strength across countries**.

---

## 📊 Asset Distribution

Three donut charts display how military assets are distributed globally:

* ✈️ **Aircraft by Country**
* 🛡 **Tanks by Country**
* 🚢 **Naval Fleet by Country**

These charts show how different nations contribute to the total global military assets.

---

## 🎛 Interactive Features

The dashboard includes interactive elements such as:

* **Country Filter Dropdown** – View data for specific countries
* **Dynamic Visual Updates** – Charts update based on selections
* **Cross Filtering** – Selecting a chart filters other visuals
