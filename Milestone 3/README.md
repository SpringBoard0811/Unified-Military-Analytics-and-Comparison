# 📊 Milestone 3 – Dashboard Development

## Unified Military Analytics and Comparison Dashboard

Milestone 3 focuses on converting the engineered dataset and KPIs into interactive visual dashboards for military analysis and country comparison using Power BI.

---

## 🎯 Objective of Milestone 3

The goal of this milestone is to build dashboards that allow users to:

- Explore global military statistics visually  
- Analyze individual country capabilities  
- Compare military strength between countries  
- Understand insights derived from KPIs

## ⚡ Quick Stats

The **Quick Stats** dashboard provides a global snapshot of military strength across all countries in the dataset. It is designed to give users an immediate understanding of rankings, distributions, and key performance indicators.

**Visuals and Components**

- KPI cards showing:
  - Total countries analyzed
  - Highest defense budget
  - Highest manpower
  - Best Power Index rank
- Bar chart of Top 10 countries by Power Index
- Defense budget comparison chart
- Asset distribution visuals (air, land, naval)
- Slicers for Region, Continent, and Alliance

**Purpose**

This dashboard helps users quickly identify global leaders and overall military trends before diving into detailed analysis.

---

## 🌍 Nation Overview

The **Nation Overview** dashboard focuses on an in-depth military profile of a single selected country.

**Visuals and Components**

- Country selector slicer
- KPI cards for:
  - Power Index Rank Gap
  - Assets per Capita
  - Budget-to-GDP Ratio
- Bar/Radar charts for:
  - Aircraft inventory
  - Naval assets
  - Tanks and land forces
  - Military manpower
- Defense budget visualization
- Tooltip comparisons showing how the country ranks globally

**Purpose**

This dashboard allows users to deeply analyze the strengths, weaknesses, and strategic position of any individual country.

---

## ⚔️ Compare Powers

The **Compare Powers** dashboard enables side-by-side comparison of two selected countries.

**Visuals and Components**

- Dual country selectors
- Comparative bar charts for:
  - Military personnel
  - Aircraft and fighters
  - Naval fleet
  - Armored vehicles
  - Defense budget
- KPI comparison cards
- Visual gap indicators showing which country leads in each metric

**Purpose**

This dashboard helps users understand relative strengths and weaknesses between two nations through clear visual benchmarking.
