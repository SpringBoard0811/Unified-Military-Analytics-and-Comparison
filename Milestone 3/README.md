# 📊 Milestone 3 – Dashboard Development

## Unified Military Analytics and Comparison Dashboard

Milestone 3 focuses on converting the engineered dataset and KPIs into interactive visual dashboards for military analysis and country comparison using Power BI.

---

## 🎯 Objective of Milestone 3

The goal of this milestone is to build dashboards that allow users to:

- Explore global military statistics visually  
- Analyze individual country capabilities  
- Compare military strength between countries  
- Understand insights derived from KPIs  

---

## 📂 Files in this Milestone

```text
ComparePowers.pbix
NationOverview.pbix
QuickStats.pbix
dashboards pdfs
```
## 📊 Dashboards Developed

⚡ Quick Stats

A high-level overview dashboard showing global military trends and rankings.

Top countries by key military indicators
Defense budget comparison
Summary KPI cards
Snapshot of global military power distribution

🌍 Nation Overview

A detailed dashboard focused on the military profile of a selected country.

Military manpower statistics
Air, land, and naval capability metrics
Defense budget insights
KPI-based evaluation for the selected nation

⚔️ Compare Powers

A comparative dashboard allowing side-by-side analysis of two countries.

Comparison of personnel, aircraft, naval assets, and armored units
KPI comparison between selected countries
Visual benchmarking of strengths and weaknesses
