# 🎯 Milestone 3 – Full Dashboard Development

## 📌 Overview

Milestone 3 focuses on transforming the KPI-engineered dataset into a fully interactive Power BI dashboard system. This stage builds upon the dataset created in Milestone 2 and implements advanced visual analytics, filtering, and scenario simulations to analyze global military power.

The goal is to create a professional-grade analytical dashboard that allows users to explore military capabilities, compare countries, and simulate coalition strengths.

The dashboard enables users to:

* Explore military capabilities of individual countries
* Compare multiple nations side-by-side
* Simulate coalition strength by combining military resources

The system consists of **four interconnected dashboards** designed for exploratory and comparative analysis.

---

# ⚡ Quick Stats & Nation Overview

## Objective

Provide a **high-level global overview** of military strength while allowing users to explore **detailed country-level profiles**.

## Key Features

### 📊 Quick Stats Dashboard

* Displays **Top 10 countries by Power Index**
* Provides a **global overview of military capabilities**
* Interactive **filters for:**

  * 🌍 Region
  * 🌎 Continent
  * 🤝 Alliance

### 📌 KPI Cards

Key indicators are displayed as quick summary metrics:

* 🥇 Power Index Rank
* 💰 Defense Budget
* ✈️ Aircraft Count
* 👥 Military Personnel

### 🌍 Nation Overview

Users can select a country and view its **complete military profile**, including:

* Military rank and global standing
* Defense budget
* Personnel and manpower
* Air, naval, and land assets

### 📈 Visualizations

Data is displayed using multiple visualization types:

* **Bar Charts** – asset comparison
* **Radar Charts** – capability distribution
* **Comparative Charts** – country vs global averages

### 🧠 Interactive Tooltips

Tooltips provide additional context such as:

* Global rankings
* Regional comparisons
* Key KPI insights

# ⚔️ Compare Powers & Coalition Builder

## Objective

Allow users to **compare countries and simulate coalition strength** through interactive selections.

---

## ⚔️ Compare Powers Dashboard

### Features

* Side-by-side comparison of **two selected countries**
* Direct comparison of major military indicators:

| Metric            | Description                        |
| ----------------- | ---------------------------------- |
| 👥 Manpower       | Total active and reserve personnel |
| ✈️ Aircraft       | Air power capability               |
| 🚢 Naval Assets   | Naval strength                     |
| 💰 Defense Budget | Military spending                  |
| 📊 KPIs           | Derived analytical indicators      |

Visual comparisons highlight **relative strengths and weaknesses**.

---

## 🤝 Coalition Builder Dashboard

### Features

Allows users to **select multiple countries** and simulate a coalition.

The dashboard automatically calculates **combined military power**, including:

* Total manpower
* Total aircraft
* Total naval assets
* Combined defense budget

### Coalition Insights

Users can evaluate:

* Potential strategic alliances
* Regional military balance
* Combined power of selected nations
