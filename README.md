# 🌍 Unified Military Analytics and Comparison Dashboard

The **Unified Military Analytics and Comparison Dashboard** is a data analytics project designed to transform global military data into meaningful insights through interactive visualization.  
By combining **data processing, KPI engineering, and dashboard development**, the project enables users to analyze military capabilities, compare countries, and explore strategic defense insights through an intuitive analytics platform.

The system focuses on making complex defense data easier to understand by presenting it through **structured datasets and interactive dashboards**.

---

# 📌 Project Overview

This project analyzes and visualizes **global military capabilities across countries** using modern data analytics and visualization techniques.

The dashboard allows users to:

- 🌐 Explore global military rankings  
- 🏳️ Analyze country-level military strength  
- ⚖️ Compare military capabilities between nations  
- 🤝 Simulate coalition military power  

The system integrates **data preprocessing, KPI engineering, and interactive dashboards** to provide a powerful platform for military analytics.

---

# 🎯 Project Objectives

- 📊 Collect and process global military capability data

# 🧠 Key Performance Indicators (KPIs)

To enhance analytical insights, the following KPIs were created:

| KPI | Description |
|----|-------------|
| 📈 **Power Index Rank Gap** | Difference in ranking between countries |
| ⚙️ **Assets per Capita** | Military assets relative to population |
| 💰 **Budget-to-GDP Ratio** | Defense spending relative to economic output |
| 👥 **Personnel Ratio** | Military personnel relative to population |

These KPIs help evaluate **military efficiency, economic investment, and resource distribution**.

# 📊 Dashboard Modules

The analytics system contains **four interactive dashboards**.

---

## 🌐 Quick Stats Dashboard

Provides a **global overview of military power**.

### Features

- Top countries by **Power Index**
- Global military rankings
- Defense budget comparison
- Interactive filters by region and alliance

---

## 🏳️ Nation Overview Dashboard

Provides **detailed insights into a selected country's military strength**.

### Metrics Visualized

- Military personnel
- Aircraft count
- Naval fleet strength
- Defense budget
- Power Index score

---

## ⚖️ Compare Powers Dashboard

Allows **side-by-side comparison between two countries**.

### Comparison Metrics

- Military personnel
- Aircraft assets
- Naval strength
- Defense spending
- KPI indicators

---

## 🤝 Coalition Builder Dashboard

Simulates **combined military strength for alliances**.

### Coalition Analysis

Users can analyze:

- Total personnel
- Combined aircraft
- Naval fleet size
- Total defense budget

This helps evaluate **strategic military alliances**.
