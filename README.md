# 🌍 Unified Military Analytics & Comparison Dashboard

This project presents an interactive **global military analytics platform** built to analyze and compare military capabilities of **140+ countries** using open-source defense data.

The system uses **Python-based data pipelines** to scrape, clean, and process military metrics from GlobalFirepower.com and convert them into meaningful analytical insights.

The processed data is then visualized through **interactive dashboards**, enabling users to explore global military strength, compare nations, and analyze strategic indicators.

---

# 🎯 Project Purpose

The primary objective of this project is to build a **data-driven military analytics system** that helps users:

* Analyze global military rankings
* Compare military capabilities between countries
* Understand defense spending and economic relationships
* Simulate the combined strength of military alliances

To support deeper insights, the project introduces custom analytical KPIs such as:

* **Power Index Rank Gap**
* **Assets per Capita**
* **Defense Budget to GDP Ratio**

---

# 🌐 Data Source

All military statistics used in this project are collected from:

**GlobalFirepower.com**

Data is scraped using a predefined list of country URLs and converted into structured datasets for analysis.

### Dataset Highlights

* 🌍 Coverage of **140+ countries**
* 📊 More than **50 military and economic indicators**

Examples of collected metrics include:

* Total manpower
* Aircraft inventory
* Naval assets
* Tanks and armored units
* Defense budgets
* Economic indicators such as GDP

# 📊 Dashboard Components

The project contains four interactive dashboard modules designed to analyze military capabilities from multiple perspectives.

### ⚡ Quick Stats

Provides a high-level overview of global military rankings and key indicators.

Main features:

* Top 10 countries by military strength
* Global ranking insights
* Region and alliance filters
* Key KPI indicators

---

### 🌍 Nation Overview

Displays a detailed military profile for an individual country.

Includes:

* Military manpower statistics
* Air, land, and naval capabilities
* Defense budget analysis
* Derived analytical KPIs

---

### ⚔️ Compare Powers

Allows users to compare **two countries side-by-side**.

Comparison includes:

* Personnel strength
* Aircraft inventory
* Naval assets
* Defense budgets
* Strategic KPIs

---

### 🤝 Coalition Builder

Simulates the **combined military strength of multiple countries**.

Capabilities:

* Multi-country selection
* Aggregated military statistics
* Coalition vs individual country comparison
