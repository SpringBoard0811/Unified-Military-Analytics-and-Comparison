# 🌍 Unified Military Analytics and Comparison Dashboard

An interactive analytics platform designed to explore and compare global military power using open-source defense data.

The system collects, processes, and visualizes military capability indicators for **140+ countries** using Python-based data pipelines and interactive dashboards.  
The goal of this project is to transform raw defense data into meaningful insights that allow users to understand and compare global military strength.

---

## 🎯 Project Objective

This project aims to build a comprehensive analytics system that enables users to analyze and compare military capabilities of different nations.

Users can:

✔ Explore global military rankings  
✔ Compare military capabilities between countries  
✔ Analyze defense resources and strategic indicators  
✔ Visualize military metrics using interactive dashboards  

Key analytical metrics include:

- Power Index Rank Gap  
- Assets per Capita  
- Defense Budget to GDP Ratio  

---

## 🛰 Data Source

Military statistics used in this project are collected from open-source defense data.

Primary Source:

🌐 **GlobalFirepower**

The dataset includes information such as:

- Military manpower  
- Aircraft inventory  
- Naval fleet strength  
- Defense budget statistics  
- Economic indicators related to defense capability  

Dataset Coverage:

- **140+ countries**
- **50+ military and economic indicators**

## 📊 Dashboard Modules

The platform integrates more than **50 defense and economic indicators** and organizes them into four analytical modules.

• **Global Snapshot** – Overview of worldwide military rankings and key trends  

• **Country Insights** – Detailed analysis of military capabilities of an individual nation  

• **Power Comparison** – Side-by-side comparison of two countries across multiple military indicators  

• **Alliance Simulator** – Evaluation of combined military strength of multiple countries
