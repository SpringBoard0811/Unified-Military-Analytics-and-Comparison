# 📌 Milestone 2 – KPI Engineering & Dashboard Planning

## 🔍 Overview

The objective of this milestone is to enhance the cleaned dataset by generating meaningful KPIs and preparing the structure for interactive dashboard development.

## ⚙️ Key Activities

Identified key performance indicators (KPIs) required for analysis, including Power Index Rank Gap, Assets per Capita, and Defense Budget-to-GDP Ratio.
Calculated KPIs using the cleaned dataset.
Enriched the dataset by adding additional information such as region, continent, and alliance categories.
Transformed the dataset into formats suitable for visualization tools like Tableau and Power BI.
Designed the initial dashboard structure, including modules like Quick Stats, Nation Overview, Compare Powers, and Coalition Builder.
