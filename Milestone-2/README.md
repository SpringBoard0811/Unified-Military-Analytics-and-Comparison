# 📊 Milestone 2: Data Cleaning & KPI Engineering

### 🧠 Transforming Raw Data into Actionable Intelligence

---

## 🧠 Overview

After collecting raw data, this phase focuses on converting it into a **clean, structured, and analytics-ready dataset**.

Additionally, advanced **Key Performance Indicators (KPIs)** are engineered to extract meaningful insights from the data.

---

## 🎯 Objectives

* Clean and standardize raw data
* Handle missing and inconsistent values
* Convert data into numeric and usable format
* Generate analytical KPIs for insights

---

## 🧹 Data Cleaning Pipeline

### 🔍 Data Preprocessing Steps

* Removed unwanted symbols (`%`, `+`, `,`)
* Converted string values into numeric format
* Standardized column naming conventions
* Normalized inconsistent units

---

### ⚙️ Data Validation

* Identified null values
* Applied imputation techniques
* Ensured no duplicate entries
* Verified data consistency

---

## 📊 Feature Engineering

### 🧠 KPI Development

The following KPIs were engineered to enhance analytical capability:

---

### 🏆 Power Index Rank Gap

* Measures relative difference between countries
* Helps identify strength disparities

---

### 👥 Assets per Capita

* Ratio of military assets to population
* Indicates resource efficiency

---

### 💰 Budget-to-GDP Ratio

* Measures defense spending intensity
* Highlights economic commitment to defense

---

### ⚙️ Composite Strength Metrics

* Aggregated indicators for overall military capability

---

## 🔄 Data Transformation
