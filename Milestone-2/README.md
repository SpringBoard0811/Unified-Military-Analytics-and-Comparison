# Milestone 2: KPI Engineering and Dashboard Preparation

## Overview

Milestone 2 focuses on transforming the cleaned military dataset into a more analytical format by generating meaningful Key Performance Indicators (KPIs). These KPIs help evaluate the relative military strength of different countries and prepare the data for visualization in dashboard tools such as Power BI or Tableau.

## Objectives

* Create calculated metrics to enhance analysis
* Enrich the dataset with additional attributes
* Prepare structured datasets for visualization

## Tasks

* Calculate important KPIs such as:

  * Power Index Rank Gap
  * Assets per Capita
  * Defense Budget to GDP Ratio
* Add metadata such as:

  * Region
  * Continent
  * Alliance information (for example NATO)
* Convert datasets into formats suitable for dashboard tools

## Deliverables

* `military_final.xlsx`
* Python script: `generate_kpis.py`

## Expected Outcome

A final dataset containing enriched military indicators and calculated KPIs that can be directly used for building dashboards and performing comparative analysis between countries.
